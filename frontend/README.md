# Gestor de Tareas Fullstack

Aplicación fullstack para la gestión de tareas con sistema de recomendaciones básicas.  
Desarrollada como proyecto personal, lo básico para práctica.

---

## Demo

Proyecto en desarrollo, en local:
- Frontend: http://localhost:3000
- Backend: http://localhost:3001

---

## Funcionalidades

- CRUD

---

### Frontend
- React
- JavaScript (ES6)
- Fetch API
- CSS inline básico

### Backend
- Node.js
- Express
- SQLite
- CORS
- Body-parser
